## 🚀 Быстрый старт (Windows)

1. Перейдите в папку [`release-windows/`](release-windows)
2. Скачайте `KK.exe` и `launcher.py`
3. Поместите оба файла в одну папку
4. Запустите лаунчер: `python launcher.py`
